# Koydo Worlds — Character Design Sheet Brief
### Commission Document for Character Concept Artists

> **Project:** Koydo Worlds — UE5 educational game world system
> **Scope:** 50 Kindler Guide characters across 3 realms + hub
> **Purpose:** Approved 2D design sheets that feed (1) MetaHuman 3D creation, (2) Fal.ai portrait generation for in-game UI/menus
> **IP Terms:** Full work-for-hire, same terms as companion brief. All work must be original.
> **Reference images:** Fal.ai-generated concept portraits are attached per character (see below each entry). These are AI-generated starting points for creative reference only — the artist's design should interpret and elevate them, not copy them.

---

## SECTION 1: ART DIRECTION & STYLE GUIDE

### Visual Identity

Koydo Worlds is a hyper-realistic educational world system rendered in UE5. Characters are rendered via Epic MetaHuman at T4 tier — photorealistic, emotionally nuanced, fully animated. The art direction is:

- **Grounded realism with personality** — characters look like real people (or real fantastical beings), not cartoons. Proportions are realistic except for intentionally non-human characters (Cal, Atlas, Baxter, etc.)
- **Cultural specificity** — every character's cultural background should be visible and respectfully realized, not generic
- **Readable silhouette** — each character must be instantly recognizable from a distance by their silhouette alone
- **Distinctive detail** — each character has one signature detail that children remember (Zara's brass prosthetic arm, Oliver's coral-tipped cane, Yuki's precisely-cut hair)

### Consistent Art Style Across All 50 Characters

| Parameter | Standard |
|-----------|----------|
| Rendering style | Semi-realistic concept art — detailed enough for MetaHuman reference, expressive enough for emotional reads |
| Line weight | Clean, confident, medium weight. No scratchy sketches. |
| Color | Full color. Each character has a dominant color associated with their world (see per-character notes) |
| Lighting | Soft natural 3/4 rim light — designed to show face and form clearly, not dramatic |
| Background | Subtle environmental suggestion of their world (no full backgrounds needed) |

### Character Tier Reference

| Tier | Who | Production use |
|------|-----|----------------|
| T4 (Hero) | All 50 Kindler Guides | Full MetaHuman rig, facial animation, strand hair, RigLogic muscle sim |
| T3 | Named secondary NPCs | MetaHuman base + dialogue system |
| T2 | World-specific background NPCs | Fal.ai portrait only |
| T1 | Crowd | Procedural generation |

**All 50 characters in this brief are T4 heroes.**

## SECTION 2: DESIGN SHEET SPECIFICATION

### What to Deliver Per Character

Each character requires a single design sheet containing:

1. **Front view** (full body, neutral pose, no expression) — primary reference for MetaHuman
2. **3/4 view** (face turned slightly, full body) — secondary reference
3. **Expression sheet** — minimum 4 expressions: `neutral`, `teaching/engaged`, `delighted`, `concerned/serious`
4. **Detail callout** — 2–3 zoomed details of signature elements (prosthetic arm, specific tool, distinctive clothing, cultural jewelry, etc.)
5. **Color palette strip** — skin tone, hair, primary outfit colors, accent colors

### Sheet Format

- **Canvas:** A3 landscape (4961×3508px at 300dpi) or equivalent
- **File formats:** PSD source + flattened PNG
- **Naming:** `kindler-[slug].psd` (e.g., `kindler-professor-nimbus.psd`)

### Expression Sheet Minimum

| Expression | Notes |
|------------|-------|
| `neutral` | Resting face, present but not performing |
| `teaching` | Animated, explaining, leaning slightly forward — this is their default mode |
| `delighted` | Full genuine happiness — a child just understood something |
| `concerned` | When a child is struggling — empathetic, not stern |
| `world-specific` (bonus) | One expression unique to their character (Yuki's data-recognition stare, Atlas's geological patience, etc.) |
