# Prompt Library

Reusable prompt briefs for coding agents, research agents, launch operators, design teams, and executive review workflows.

## Folder Map
- `docs/prompts/orchestration/`
  - Prime orchestrators, executive-council prompts, and cross-portfolio launch review briefs.
- `docs/prompts/audit/`
  - Inspection prompts for launch gates, quality reviews, compliance checks, and hardening passes.
- `docs/prompts/creation/`
  - Creation prompts for UI, landing pages, listings, and campaign production.
- `docs/prompts/strategy/`
  - Market, finance, partnerships, investor, and corporate-strategy prompts.
- `docs/prompts/content-production/`
  - Production briefs for narrations, media packs, and content-generation handoffs.

## Start Here
- Use [INDEX.md](D:/PythonProjects/Koydo/docs/prompts/INDEX.md) to choose the best prompt for a task.
- Use the files directly by path.
- Treat these prompts as reusable operator briefs, not source-of-truth documentation.
- Prompts must verify implementation against real code and config.
